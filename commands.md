# Docker Commands:

| Task                                   | Command                            |
| -------------------------------------- | ---------------------------------- |
| Build                                  | docker build <path_to_dockerfile>  |
| Run and publish a port on a local port | docker run -p 3000:3000 <image_id> |
| List running containers                | docker ps                          |
| Stop a running container               | docker stop <name>                 |
